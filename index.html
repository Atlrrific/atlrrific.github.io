Beathoven

AI Song Composer in python

Hack The Planet Submission

Developers : Atl Arredondo and Anthony Lowhur

The whole project is divided into three parts: 1) Signal processing - Extracting various features of sound like frequency domain, time domain, spectral domain, etc and segmenting them individually. 2) Machine learning - Feeding the segmented data to neural networks and creating different variations of the same original input. 3) Sound synthesis - Combining the segmented data to create new unique music.

We used Fast Foruier transforms in order to extract the frequencies from .wav music files and Long Short Term Memory Neural Netwroks to create the music frequencies that made up the song. Enjoy

Fixey - A transportation platform which mixed bike and public transportation to provide the greeniest and healthiest route for a commuter.

Cal Hacks Best Health-Tech Hack Winner

Developers : Atl Arredondo, Henry Millison, Anish Kannan and Adrian Galicia

As people who are avid bikers, we were frustrated by the inability of Google Maps to combine public transit and biking directions together to give you the most efficient way to get from point a to point b. Through Fixey, you can get the best path by combining transit directions with biking directions if a bike will provide a more efficient way. We used the Google Maps API to get the directions along with Leaflet, Mapbox and D3.JS to provide an appealing visualization of your results.

Project Tron - Play Tron on google glass in real time by using your position to defeat your opponents

Developers : Atl Arredondo, Daniel Dao and Andrew Xia

Finally after years of technology innovation, you can play Tron on real time using your location to leave a track behind and crush your opponents. Project Tron tracks down the gps location and display the game interface on Google Glass using a mapbox map on android. The tracking data from the different players get mapped on lines and the app constantly checks for player collisions by pushing and popping data from a Mongo DB database. This app is made for use on bicycles since its easier to play and more fun. Have fun my friends.

Project X - An Interactive Graph to learn about Reseach Projects

Developers : Atl Arredondo, David Desimone, Ram Hari Dahal and Jose Ramirez

This is a graph theory research project where we created an interactive graphs where each node is a Research Experience for Undergrduates(REU) Projects connected by their similarities using their Jaccard Index. The user can interact with the graph by clicking and hovering on the card abstraction of the project to find out more about the details of the reserach. This with the purpose to motivate the user to learn more about STEM and the research projects that involve these disciplines.
